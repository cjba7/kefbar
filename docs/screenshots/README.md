# Screenshots

The main README references two images here. Drop them in with these exact names:

- **`panel.png`** is the menu-bar panel. Click the speaker icon in the menu bar to open
  it, then capture just the panel. Light mode with the Dark Olive palette looks best as
  the hero.
- **`settings.png`** is the Settings window. The **UI** tab shows off the palettes and
  appearance options nicely; the **Speakers** tab is also a good choice.

PNG, retina (2x) is fine. The README sizes them down.
